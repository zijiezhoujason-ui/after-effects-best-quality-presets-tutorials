# After Effects Presets & Tutorials

This repository contains my personal After Effects presets, motion design experiments, and tutorial notes.
I created these resources to share useful editing techniques, improve my workflow, and document how different visual effects are built from scratch.

## About This Project

I am interested in video editing, motion graphics, and visual storytelling.
Through this project, I upload After Effects presets and explain the creative and technical process behind them.

The goal of this repository is not only to share finished effects, but also to show how they work, why certain settings matter, and how other editors can customize them for their own videos.

## Contents

### Presets

The `presets` folder includes reusable After Effects preset files, such as:

* Camera shake effects
* Smooth transitions
* Text animation presets
* Color grading presets
* Glow and blur effects
* Sports highlight editing effects

### Tutorials

The `tutorials` folder includes step-by-step explanations for each effect.
Each tutorial explains:

* What the effect does
* When to use it
* Required tools or plugins
* Step-by-step setup
* Important settings
* Customization tips

### Previews

The `previews` folder includes GIFs or images showing what each preset looks like before downloading.

## How to Use the Presets

1. Download the preset file from the `presets` folder.
2. Open Adobe After Effects.
3. Import or apply the preset to your layer.
4. Adjust the settings based on your footage.
5. Check the related tutorial for customization tips.

## Featured Tutorials

### 1. Basic Camera Shake Effect

A simple camera shake preset designed for sports edits, gaming clips, and impact moments.
It can be used to make hits, dunks, kills, or beat drops feel more powerful.

### 2. Smooth Transition Preset

A transition effect designed to create fast but clean movement between clips.
It works well for highlight videos, short-form edits, and social media content.

### 3. Cinematic Text Animation

A text animation preset for intros, title cards, and dramatic edits.
It uses scale, opacity, blur, and easing to create a cleaner motion design style.
